Conference Management Tool

Managing things manually is a hectic task. Imagine what happens when the same problem arises in a college to evaluate and comment on hundreds of assignments? Isn’t this huge? Similarly, organizing academic events such as presentations, conferences etc., manually is a huge task. This requires more man power for communicating with the participants one after the other. These are even more complicated than the assignments which is a time-consuming process. Earlier, the idea of submitting these papers or academic events through an electronic mail(e-mail) decreased some strain but, commenting and giving feedback to the participant is still a problem. The data storage of all the files and handling those files manually is also a major problem. Therefore, we proposed a web-based application i.e., the Conference Management Tool which allows admin/Organizer to create new academic events, registration of the events and allocating judges for the events. On the user side, this tool allows participant to enroll for the event and upload documents for the events. The system must make an Excel file including the details of the participants based on the submissions to the respected event. Here judge is also a user which can access the assigned poster to him to preview the details of the project and commenting on it , so that it becomes easier for him to remember the projects on the event day. Technologies Used:HTML5, CSS3, Bootstrap, PHP, JavaScript, jQuery, Ajax, MySQL
